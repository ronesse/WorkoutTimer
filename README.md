# WorkoutTimer

Mobiltilpasset intervalltimer for kettlebell.

## Standardøkt

- 20 runder
- 60 sekunder arbeid
- 30 sekunder hvile
- Total varighet: 30 minutter
- Gul bakgrunn og «Hold ut!» de siste 10 sekundene av arbeid
- «Gjør klar!» de siste 5 sekundene av hvile
- Pip, norsk tale og vibrasjon
- Lokal spilleliste fra telefonens lydfiler

## Filer

- `index.html`
- `style.css`
- `app.js`

Åpne `index.html` i en nettleser, eller publiser prosjektet med GitHub Pages.
